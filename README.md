## 1 Общее описание системы

HumanGuard — это SaaS-решение для защиты веб-сайтов от автоматизированных атак и ботов. Система работает как обратный прокси, анализируя поведение посетителей и HTTP-запросы для выявления аномальной активности. Предоставляет администраторам панель управления для настройки параметров защиты, просмотра статистики и управления заблокированными сессиями.

### Описание системы

Система состоит из следующих модулей:

- **Модуль сбора поведенческих данных** — сбор клиентских метрик на стороне браузера (движения мыши, клики, скроллинг, скорость набора текста, fingerprint)
- **Модуль анализа паттернов обращений** — анализ HTTP-запросов на серверной стороне (частота запросов, User-Agent, временные паттерны, headless-браузеры)
- **Модуль детекции аномалий** — присвоение каждой сессии оценки риска (0–100)
- **Модуль реагирования** — настраиваемые действия в зависимости от уровня риска (пропуск, CAPTCHA, блокировка)
- **Модуль логирования и мониторинга** — запись всех событий в базу данных, визуализация в админ-панели
- **Модуль управления настройками** — настройка порогов срабатывания, включение/выключение подсистем
- **Модуль аутентификации** — регистрация по email с подтверждением, 2FA через TOTP, восстановление пароля

>Подробное описание системы: `docs/SYSTEM_SPECS.md`

## 2 Функциональное назначение

### Описание внешних функций (API)
> Документ: `docs/api/README.md`.

Основные эндпоинты:

| Метод | Путь | Назначение |
|-------|------|-----------|
| POST | `/api/users` | Регистрация пользователя |
| POST | `/api/login` | Вход в систему |
| GET | `/api/sites` | Список сайтов пользователя |
| POST | `/api/sites` | Создание сайта |
| PUT | `/api/sites/{id}/settings` | Обновление настроек сайта |
| POST | `/api/sessions` | Создание сессии |
| GET | `/api/sites/{id}/stats` | Статистика по сайту |
| POST | `/api/sessions/{id}/block` | Блокировка сессии |

#### Описание ошибок 

| Код | Описание |
|-----|----------|
| 400 | Неверный запрос |
| 401 | Неавторизован / неверные учётные данные / неверный TOTP |
| 403 | Доступ запрещён |
| 404 | Ресурс не найден |
| 409 | Конфликт (email уже занят) |
| 413 | Файл слишком большой |
| 415 | Неподдерживаемый тип файла |
| 500 | Внутренняя ошибка сервера |
