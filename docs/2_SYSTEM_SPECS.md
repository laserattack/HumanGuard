# Описание системы
## Структура ПО
> Перечень программных компонентов (разрабатываемых и используемых) с указанием роли в системе. 

> [!note] Пример
> 
> 1. Nginx v1.2.3.
> 2. Веб-сервер, написанный на языке C# и фреймворке ASP.NET
> 3. Сервер обработки ML-задач, написанный на языке Python и фреймворке Django.
> 4. СУБД хранения данных PostgreSQL v13.
> 5. Брокер сообщений RabbitMQ.

## Структурная схема ПО в нотации C4
> Высокоуровневая (High-level design, HLD) диаграмма связи программных компонентов  в нотации C4. Должна быть спроектирована на уровнях Контекстов (System context) и Контейнеров (containers). 

> [!WARNING] Инструментарий
> Рекомендуется использовать Draw.io, excalidraw или Visio. Все инструменты, имеющие плагины для отрисовки C4: https://c4model.tools/

### Уровень контекста
> [!note]  Пример
> 
> https://c4model.com/diagrams/system-context

### Уровень контейнеров
> [!note] Пример
> 
> https://c4model.com/diagrams/container

## Схемы баз данных (ERD) 
> Проект используемых реляционных баз данных в нотации Entity-Relation.

> [!WARNING]  Инструментарий
> Необходимо использовать стандартные дополнения ERD к Draw.io, excalidraw или Visio.

> [!note] Пример
> 
> https://c4model.com/diagrams/container
