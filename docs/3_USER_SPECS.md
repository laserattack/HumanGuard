# Руководство пользователя

> Описание порядка взаимодействия пользователя поддерживаемых ролей (пользователь, администратор) с системой. 
> Требования к структуре руководства пользователя по серии ГОСТ 34 устанавливаются ГОСТ Р 59795 - 2021.
# Описание операций, доступных пользователю
> Для каждой операции необходимо описать:
> - наименование;
> - условия, при соблюдении которых возможно выполнение операции;
> - подготовительные действия;
> - основные действия в требуемой последовательности;
> - заключительные действия.


> [!note] Пример
> 
> 1. Наименование:
> Авторизация пользователя в системе.
> 2. Условия выполнения:
> - Администратор создал учетную запись пользователя.
> - Отсутствуют ошибки 4xx и 5xx доступа к системе
> 3. Подготовительные действия:
> - Авторизация пользователя в стороннем почтовом клиенте для получения OTP-кодов авторизации.
> 4. Основные действия:
> - перейти в браузере по URL /auth, как показано на рисунке.
> - …
> 5. Заключительные действия: 
> При успешной авторизации в правом верхнем углу интерфейса будет показано имя пользователя.
